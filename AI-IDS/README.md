# AI-Powered Intrusion Detection System

## Overview
A Proof-of-Concept (PoC) IDS that uses Machine Learning to detect network anomalies that traditional signature-based systems might miss.

## Tech Stack
*   **Python**: Core logic.
*   **Scikit-Learn**: Machine Learning library (Isolation Forest).
*   **Pandas**: Data manipulation.
*   **Streamlit**: Web dashboard for visualization.

## Features
*   Ingests NSL-KDD dataset for training.
*   Real-time anomaly scoring.
*   Visualizes attack distribution.
